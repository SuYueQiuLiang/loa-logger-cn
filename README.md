### 前排提示
* 截止目前的美服/韩服插件，以及未来的国服插件都是使用npcap进行抓包，并且分析网络包的产物，理论上不会注入游戏程序，这样能够避免绝大多数情况下的非法软件审查。
* 目前来说，抓包形式的插件因为非侵入式，常规情况下不会直接导致封号问题出现。
* 如果插件被认定为禁止使用的外部程序,tp仍然有能力封禁使用的玩家账号
